Beyond the basics of splitting, Evere has features that improve everyday convenience.

## Categories

Classify expenses by category, such as "Food" or "Transport." Several categories are provided, and you can also **add your own**.

- From "Manage categories" in the Settings tab, add, edit, or delete categories with an emoji and a name (up to 20 characters).
- Categories are **shared across all projects**.
- On the settlement screen, you can see a breakdown by category.

## Budgets (Pro)

Set a **budget** for a project and get notified when usage reaches a level you choose.

- Set the budget amount (in the **base currency**).
- As **threshold notifications**, add multiple conditions by percentage (e.g., 80%) or amount (e.g., 50,000). You're notified when a level is reached.
- On the settlement screen, check the budget and how much has been used.

:::note
Only the owner can set a budget. Budgets are a Pro feature.
:::

## Notifications

Receive push notifications about members' actions and project status. Toggle them on or off from "Notifications" in the Settings tab.

- **Expense notifications** — an expense added, updated, or deleted
- **Settlement notifications** — settled/unsettled at the expense, project, or final-settlement level
- **Budget alerts**

:::note
To use push notifications, allow notifications for Evere in your iPhone settings.
:::

## Offline (Pro)

You can log expenses even without a connection, and they **sync automatically** when you're back online. Reassuring when your connection is spotty on a trip.

- From "Offline sync" in the Settings tab, check your online status and pending changes, and sync manually if needed.

:::note
Offline use is a Pro feature. Note that some actions — creating or deleting a project, creating an invite link, changing settlement status, and managing your plan — can only be done while online.
:::
