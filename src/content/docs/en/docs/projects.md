Projects are the heart of Evere. You create one per scene — a trip, dating, shared living — and log expenses inside it.

## Creating a project

Create one from "Create a new project" in the "Projects" tab.

- **Project name** (up to 20 characters)
- **Currencies to use** — you can pick several. For a trip, it's handy to include both the local currency and your own.
- **Base currency** — one of the currencies you picked. Settlement is calculated in it. A suggestion is offered automatically based on your device's language and region.
- **Participants** (optional) — add them by name. You can also add them later.

:::note
Creating a project requires an internet connection.
:::

:::caution
**On the Free plan, you can create up to 3 projects.** Upgrade to Pro to create an unlimited number.
:::

## The active project

Selecting a project in the list makes it the **active project**. The "Expenses" and "Settlement" tabs show this active project. To work on a different project, select it in the list.

## Changing project settings

From the project settings, you can change the following (many actions are limited to the project **owner**).

### Project name

Change it anytime (up to 20 characters).

### Currency settings

- **Available currencies** — add or remove anytime.
- **Base currency** — the basis for settlement calculations.

:::caution
**Once you log the first expense, the base currency can no longer be changed.** This keeps settlement consistent. After that, you can only add or remove available currencies. To use a different base currency, create a new project.
:::

### Managing participants

Add participants, rename them, or remove them. Participants can be selected when logging an expense.

:::note
A participant who is already included in an expense can't be removed. First remove them from that expense, or edit/delete the expense.
:::

## Marking a project as settled

When everything is settled, you can mark the whole project as **settled**. After that, no new expenses can be added to it.

If you want to add expenses again, **unsettle** the project to make it editable once more.

## Deleting a project

You can delete projects you no longer need.

:::caution
Only the owner can delete, and only while online. **Deletion cannot be undone.**
:::
