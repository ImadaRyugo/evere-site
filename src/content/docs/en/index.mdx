## About Evere

Evere is a mobile application designed to help couples and groups manage bill splitting and settlements effortlessly across multiple currencies.

<CardGrid>
	<Card title="Multi-Currency Support" icon="translate">
		Handle expenses in different currencies with automatic conversion
	</Card>
	<Card title="Flexible Splitting" icon="puzzle">
		Equal split, ratio-based, or manual amount entry
	</Card>
	<Card title="Project Management" icon="list-format">
		Organize expenses by projects (trips, shared living, etc.)
	</Card>
	<Card title="Settlement Tracking" icon="approve-check">
		Mark individual expenses or entire projects as settled
	</Card>
</CardGrid>
