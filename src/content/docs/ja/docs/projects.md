プロジェクトは Evere の中心です。旅行・デート・同棲など、シーンごとに 1 つ作り、その中に支払いを記録していきます。

## プロジェクトを作成する

「プロジェクト」タブの「新しいプロジェクトを作成」から作成します。

- **プロジェクト名**（20 文字以内）
- **使う通貨** — 複数選べます。旅行なら現地通貨と自分の通貨を両方入れておくと便利です。
- **基準通貨** — 選んだ通貨のうちの 1 つ。精算はこの通貨で計算されます。端末の言語・地域から自動で候補が提案されます。
- **参加者**（任意） — 名前で追加できます。あとからでも追加できます。

:::note
プロジェクトの作成はオンライン時のみ行えます。
:::

:::caution
**Free プランでは、作成できるプロジェクトは 3 個までです。** それ以上作りたい場合は Pro にアップグレードすると無制限になります。
:::

## アクティブなプロジェクト

一覧でプロジェクトを選ぶと、それが**アクティブなプロジェクト**になります。「支出」「精算」タブは、このアクティブなプロジェクトの内容を表示します。別のプロジェクトを扱いたいときは、一覧で選び直します。

## プロジェクトの設定を変更する

プロジェクト詳細の設定から、次の項目を変更できます（多くはプロジェクトの**オーナーのみ**が変更できます）。

### プロジェクト名

いつでも変更できます（20 文字以内）。

### 通貨設定

- **使用可能な通貨** — 追加・削除がいつでもできます。
- **基準通貨** — 精算計算の基準です。

:::caution
**最初の支払いを記録すると、基準通貨は変更できなくなります。** 精算の整合性を保つための仕様です。以降は使用可能な通貨の追加・削除のみ可能です。基準通貨を変えたい場合は、新しいプロジェクトを作成してください。
:::

### 参加者の管理

参加者の追加・名前の変更・削除ができます。参加者は支払いの記録時に選択できます。

:::note
すでに支払いに含まれている参加者は削除できません。先にその支払いから外すか、支払いを編集・削除してください。
:::

## プロジェクトを精算済みにする

すべての精算が終わったら、プロジェクト全体を**精算済み**にできます。精算済みにすると、そのプロジェクトには新しい支払いを追加できなくなります。

再び支払いを追加したくなったら、**精算を解除**すればまた編集できるようになります。

## プロジェクトを削除する

不要になったプロジェクトは削除できます。

:::caution
削除はオーナーのみ・オンライン時のみ行えます。**削除すると取り消せません。**
:::
