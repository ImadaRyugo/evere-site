基本の割り勘に加えて、Evere には日々の使い勝手を高める機能があります。

## カテゴリ

支払いを「食費」「交通費」などのカテゴリで分類できます。あらかじめいくつかのカテゴリが用意されているほか、**自分でカテゴリを追加**することもできます。

- 設定タブの「カテゴリ管理」から、絵文字と名前（20 文字以内）で追加・編集・削除できます。
- カテゴリは**すべてのプロジェクトで共通**して使えます。
- 精算画面では、カテゴリ別の内訳が確認できます。

## 予算（Pro）

プロジェクトに**予算**を設定し、使用状況が一定に達したら通知を受け取れます。

- 予算金額を設定します（**基準通貨**で設定します）。
- **到達通知**として、パーセント（例：80%）や金額（例：50,000）の条件を複数追加できます。指定した水準に達すると通知が届きます。
- 精算画面で、設定予算と使用済みの状況を確認できます。

:::note
予算はオーナーのみが設定できます。また、予算設定は Pro プランの機能です。
:::

## 通知

メンバーの操作やプロジェクトの状況を、Push 通知で受け取れます。設定タブの「通知」からオン・オフを切り替えられます。

- **支払い通知** — 支払いの追加・更新・削除
- **精算通知** — 支払い単位・プロジェクト全体・最終精算の完了/解除
- **予算アラート**

:::note
Push 通知を使うには、iPhone の設定で Evere の通知を許可してください。
:::

## オフライン（Pro）

電波のない場所でも支払いを記録でき、オンラインに復帰すると**自動で同期**されます。旅先で通信が不安定なときにも安心です。

- 設定タブの「オフライン同期」で、オンライン状態や保留中の変更を確認し、手動で同期することもできます。

:::note
オフラインでの利用は Pro プランの機能です。なお、プロジェクトの作成・削除、招待リンクの作成、精算状態の変更、プラン管理などの一部の操作は、オンライン時のみ行えます。
:::
