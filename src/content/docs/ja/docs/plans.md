## Free プランと Pro プラン

Evere は無料で、割り勘に必要な機能はひととおり使えます。上限をなくしたい・さらに便利にしたいときは Pro をどうぞ。

| 機能 | Free | Pro |
| --- | --- | --- |
| プロジェクト数 | 3 個まで | 無制限 |
| レシート画像 | 5 枚 / プロジェクト | 無制限 |
| マルチ通貨・自動換算 | ✓ | ✓ |
| メンバー招待・リアルタイム同期 | ✓ | ✓ |
| 手動為替レート | — | ✓ |
| 予算設定・到達通知 | — | ✓ |
| オフライン | — | ✓ |
| 広告 | 表示あり | 非表示 |

:::note
価格や購入プラン（月額・年額・買い切り）はアプリ内でご確認ください。Pro には無料トライアルが用意されている場合があります。
:::

### アップグレード・購入の復元

- アップグレードは、設定タブの「アカウント管理」または「プラン詳細」から行えます。
- 機種変更などで購入を引き継ぐときは「**購入を復元**」を使います。
- サブスクリプションは Apple ID に請求され、Apple のサブスクリプション設定からいつでも管理・解約できます。

## 困ったときは

### 精算の金額が合わない／相手が表示されない

**未確認の為替レート**がある支払いは、精算の計算から除外されます。精算画面に警告が出ていないか確認し、対象の支払いのレートを確定してください（[マルチ通貨と為替レート](/ja/docs/currencies/#未確認の為替レートに注意)）。

### 支払いを追加できない

そのプロジェクトが**精算済み**になっている可能性があります。プロジェクト一覧から精算を解除すると、再び追加できます。

### 基準通貨を変更できない

最初の支払いを記録すると、基準通貨は変更できなくなります。別の基準通貨にしたい場合は、新しいプロジェクトを作成してください（[プロジェクト](/ja/docs/projects/#通貨設定)）。

### 招待や精算ができない

これらの操作にはアカウント登録が必要です。ゲストモードでは行えません。設定タブからアカウントを登録してください（[アカウントとデータ](/ja/docs/account/)）。

### 機種変更でデータが消えた／消えそう

ゲストモードのデータは端末内のみに保存されます。機種変更の前に、必ずアカウントを登録してデータをクラウドに保存してください。

### プロジェクトを 4 個以上作りたい／広告を消したい

これらは Pro プランで解決できます（上の比較表を参照）。

### それでも解決しないとき

[お問い合わせフォーム](/ja/support/)からご連絡ください。アプリ内の設定タブ「サポート」からも問い合わせできます。
