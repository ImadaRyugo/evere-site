Evere は、カップルやグループでの割り勘・精算をストレスなく管理するためのモバイルアプリケーションです。マルチ通貨対応で、旅行や海外在住の方にも最適です。

## 主な機能

- **マルチ通貨対応**: 複数の通貨で支払いを記録し、自動換算
- **柔軟な分割**: 等分・比率・手動金額入力
- **プロジェクト管理**: 旅行や共同生活などでプロジェクトごとに整理
- **レシート添付**: レシート画像を添付して記録
- **精算管理**: 個別の支払いやプロジェクト全体を精算済みに
- **オフライン対応**: オフラインで記録し、接続時に同期（Pro）

## はじめ方

1. App Store から Evere をダウンロード（iOS）
2. アカウント作成、またはゲストモードで利用
3. プロジェクトを作成して参加者を招待
4. 支払いを追加して分割
5. 精算を追跡

## 対応プラットフォーム

- iOS 15.1 以降
