Evere（エヴァー）は、カップルやグループの割り勘・精算をかんたんに管理できる iOS アプリです。多通貨に対応し、旅行から日々の生活費まで、ひとつのアプリで完結します。

このガイドを読めば、Evere のすべての機能を使いこなせるようになります。まずは基本のしくみと画面構成から見ていきましょう。

## Evere の 4 つの基本コンセプト

Evere を理解するうえで、次の 4 つを押さえておくとスムーズです。

- **プロジェクト** — 割り勘の「単位」です。旅行・デート・同棲など、シーンごとに 1 つ作ります。支払いはすべてプロジェクトの中に記録されます。
- **基準通貨** — そのプロジェクトの精算をどの通貨で計算するかを決める通貨です。プロジェクト作成時に選びます。
- **参加者** — プロジェクトを一緒に使うメンバーです。アプリを持っている人は招待リンクから参加でき、持っていない人は名前だけで追加できます。
- **精算** — 「誰が誰にいくら払えばいいか」を Evere が自動で計算します。支払いを記録していくだけで、精算状況がいつでも確認できます。

## 画面構成（5 つのタブ）

Evere の下部には 5 つのタブがあります。

| タブ | 役割 |
| --- | --- |
| **プロジェクト** | プロジェクトの一覧・作成・切り替え・設定 |
| **支出** | 選択中プロジェクトの支払い一覧 |
| **追加** | 支払いを記録する |
| **精算** | 「誰が誰にいくら」の計算結果・精算状況 |
| **設定** | アカウント・プラン・通貨・通知などの設定 |

支出・精算タブは、**現在アクティブなプロジェクト**の内容を表示します。プロジェクトタブで別のプロジェクトを選ぶと、表示が切り替わります。

## 対応環境

- **iOS 15.1 以降**の iPhone に対応しています。
- Android 版は現在提供していません。
