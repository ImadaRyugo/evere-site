## Evere について

Evere は、カップルやグループでの割り勘・精算をストレスなく管理するためのモバイルアプリケーションです。マルチ通貨対応で、旅行や海外在住の方にも最適です。

<CardGrid stagger>
	<Card title="マルチ通貨対応" icon="translate">
		複数の通貨で支払いを記録し、自動換算
	</Card>
	<Card title="柔軟な分割" icon="puzzle">
		等分・比率・手動金額入力
	</Card>
	<Card title="プロジェクト管理" icon="suitcase">
		旅行や共同生活などでプロジェクトごとに整理
	</Card>
	<Card title="精算管理" icon="approve-check">
		個別の支払いやプロジェクト全体を精算済みに
	</Card>
</CardGrid>
