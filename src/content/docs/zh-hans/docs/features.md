除了基本的分账，Evere 还有让日常更顺手的功能。

## 类别

用“餐饮”“交通”等类别给支出分类。系统预置了若干类别，也可以**自定义添加类别**。

- 在设置标签的“类别管理”中，用表情符号和名称（20 字以内）添加、编辑或删除类别。
- 类别**在所有项目间通用**。
- 在结算界面可查看按类别的明细。

## 预算（Pro）

为项目设置**预算**，当使用量达到你设定的水平时收到通知。

- 设置预算金额（用**基准货币**设置）。
- 作为**达标通知**，可添加按百分比（如 80%）或金额（如 50000）的多个条件。达到设定水平时会收到通知。
- 在结算界面可查看所设预算和已用情况。

:::note
仅所有者可设置预算。此外，预算设置是 Pro 方案的功能。
:::

## 通知

通过推送通知了解成员的操作和项目状况。可在设置标签的“通知”中开关。

- **支出通知** — 支出的添加、更新、删除
- **结算通知** — 单笔支出、整个项目、最终结算的结算/取消
- **预算提醒**

:::note
要使用推送通知，请在 iPhone 设置中允许 Evere 的通知。
:::

## 离线（Pro）

在没有信号的地方也能记录支出，回到联网时会**自动同步**。旅途中信号不稳时也让人安心。

- 在设置标签的“离线同步”中，可查看在线状态和待处理的更改，也可手动同步。

:::note
离线使用是 Pro 方案的功能。另外，创建/删除项目、创建邀请链接、更改结算状态、管理方案等部分操作仅在联网时可用。
:::
