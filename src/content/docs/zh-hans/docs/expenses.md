在“添加”标签记录支出。无论旅行途中还是日常生活，随手把付的钱记下来就好。

## 支出的填写项

- **金额**和**货币** — 货币可从该项目的可用货币中选择。
- **付款人** — 先垫付的人（自己或参与者）。
- **参与者** — 分摊这笔支出的人。
- **分摊方式** — 从平均分摊、按比例、按金额中选择（见下文）。
- **类别** — 按餐饮、交通等分类（[类别](/zh-hans/docs/features/#类别)也可以自定义添加）。
- **标题**、**日期**和**备注** — 均为可选。
- **图片** — 可附加收据或店铺照片。

想一次记多笔时，用“**继续添加**”很方便。

### 收据等图片

在“添加图片”中，用相机拍摄或从相册选取即可附加。

:::note
免费方案每个项目最多可附加 **5 张**图片。升级到 Pro 后不限张数。
:::

### 备注

备注是自由文本框。除了简短说明，还可以在这里**保存去过的店铺链接**。

## 分摊方式

如何在参与者之间分摊一笔支出，可从三种方式中选择。

### 平均分摊

在参与者之间**平均**分。

:::note
零头会按人四舍五入，多出或少出的部分由各自承担。
:::

### 按比例

为每位参与者指定**比例（百分比）**。就像同居情侣的“6:4”，可按收入或情况来分配。总和须为 **100%**。

### 按金额

直接指定**每人应承担的金额**。系统会显示剩余金额，请让总和等于支出金额（超过总额会报错）。

## 编辑与删除支出

已记录的支出可随时编辑或删除。在“支出”标签的列表中点击对应条目打开。

## 筛选支出

在“支出”标签，可按以下条件筛选支出。

- **结算状态**（已结算 / 未结算 / 部分结算）
- **类别**
- **付款人**

## 分享单笔支出

在支出详情页可以分享这一笔。分享时可选择是否包含参与者信息和收据图片，以及是否附上以辅助货币换算的金额。
