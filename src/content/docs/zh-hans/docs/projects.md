项目是 Evere 的核心。按场景各建一个，比如旅行、约会、同居，并在其中记录支出。

## 创建项目

在“项目”标签的“创建新项目”中创建。

- **项目名称**（20 字以内）
- **要用的货币** — 可选多个。旅行时把当地货币和自己的货币都选上会很方便。
- **基准货币** — 所选货币中的一个，结算按它计算。系统会根据设备的语言和地区自动推荐。
- **参与者**（可选） — 用名字添加，之后也可以添加。

:::note
创建项目仅在联网时可用。
:::

:::caution
**免费方案最多可创建 3 个项目。** 若想创建更多，升级到 Pro 即可无限创建。
:::

## 激活的项目

在列表中选择某个项目，它就成为**激活的项目**。“支出”和“结算”标签显示这个激活的项目。要处理其他项目，请在列表中重新选择。

## 修改项目设置

在项目设置中可以修改以下项（多数操作仅限项目**所有者**）。

### 项目名称

可随时修改（20 字以内）。

### 货币设置

- **可用货币** — 可随时添加或删除。
- **基准货币** — 结算计算的基准。

:::caution
**记录第一笔支出后，基准货币就无法更改。** 这是为了保证结算的一致性。之后只能添加或删除可用货币。若想使用其他基准货币，请创建新项目。
:::

### 管理参与者

可添加参与者、修改其名字或删除。记录支出时可选择参与者。

:::note
已包含在某笔支出中的参与者无法删除。请先将其从该支出中移除，或编辑/删除该支出。
:::

## 将项目标记为已结算

当一切都结清后，可以把整个项目标记为**已结算**。之后就无法再向该项目添加新支出。

若想再次添加支出，**取消结算**即可重新编辑。

## 删除项目

不再需要的项目可以删除。

:::caution
仅所有者可删除，且仅在联网时可用。**删除后无法撤销。**
:::
