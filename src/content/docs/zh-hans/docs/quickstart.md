Evere 的基本流程只有“**创建项目 → 记录支出 → 结算**”。这里以旅行为例走一遍，大约 5 分钟。

## 第 1 步：开始使用 Evere

从 App Store 下载 Evere 并打开，然后选择开始方式。

- **用 Apple / Google / 邮箱注册** — 数据会保存到云端，邀请、记录结算等所有功能都能用。
- **以访客身份开始** — 无需注册即可立即使用，但部分功能会受限（详见[账号与数据](/zh-hans/docs/account/)）。

先设置**显示名称**。这个名字会展示给项目成员（以后可以修改）。

## 第 2 步：创建项目

在“项目”标签创建新项目。

1. 输入**项目名称**（例如“日本旅行”）。
2. 选择**要用的货币**。海外旅行时，把当地货币（如 JPY）和自己的货币（如 CNY）都选上会很方便。
3. 所选货币中的一个会成为**基准货币**，结算按这个货币计算。
4. 如有需要，用名字添加**参与者**（之后也可以添加）。

:::note
记录第一笔支出后，基准货币就无法更改了。它用于所有结算计算，因此创建项目时请仔细确认。
:::

## 第 3 步：记录支出

在“添加”标签记录支出。

1. 输入**金额**和**货币**。
2. 选择**付款人**（先垫付的人）。
3. 选择**参与者**（分摊这笔支出的人）和**分摊方式**（平均分摊、按比例或按金额）。
4. 按需添加类别、标题、日期、备注和收据图片。

用“继续添加”可以连续记录多笔。

## 第 4 步：查看结算状况

打开“结算”标签，就能看到 Evere 自动算出的结果。

- **摘要**：总支出数和总额
- **结算状况**：已结算与未结算的比例
- **支出对比**：每位参与者“付了多少（支付额）”和“该分担多少（负担额）”
- **结算**：谁该给谁多少

## 第 5 步：结算并记录

实际把钱付清后，在结算界面点击该条目，标记为**已结算**。已结算的部分会从之后的计算中排除，并记录谁已经结清。

到这里基本流程就走完了。接下来的页面会更详细地介绍各项功能。
