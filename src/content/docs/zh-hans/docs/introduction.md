Evere 是一款让情侣和团体轻松分账、结算的 iOS 应用。支持多种货币，从旅行到日常开销，一个应用就够。

跟着本指南，你就能把 Evere 的所有功能用得得心应手。先来了解核心概念和界面结构。

## Evere 的 4 个核心概念

记住这四点，其余的就容易理解了。

- **项目** — 分账的“单位”。按场景各建一个，比如旅行、约会、同居等。所有支出都记录在项目里。
- **基准货币** — 决定该项目用哪种货币结算的货币。创建项目时选择。
- **参与者** — 一起使用项目的成员。有应用的人可通过邀请链接加入，没有应用的人只填名字也能添加。
- **结算** — Evere 自动算出“谁该给谁多少”。只要记账，随时都能查看结算状况。

## 界面结构（5 个标签）

Evere 底部有 5 个标签。

| 标签 | 作用 |
| --- | --- |
| **项目** | 项目的列表、创建、切换与设置 |
| **支出** | 当前项目的支出列表 |
| **添加** | 记录支出 |
| **结算** | “谁给谁多少”的计算结果与结算状况 |
| **设置** | 账号、方案、货币、通知等设置 |

支出和结算标签显示**当前激活的项目**。在项目标签选择其他项目即可切换所显示的内容。

## 支持环境

- 支持 **iOS 15.1 及以上**的 iPhone。
- 目前暂无 Android 版。
