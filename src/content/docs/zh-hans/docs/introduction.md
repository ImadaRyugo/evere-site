Evere 是一款帮助情侣和团体轻松管理分账和结算的移动应用程序。支持多种货币。

## 主要功能

- **多货币支持**: 以不同货币记录支出并自动换算
- **灵活分账**: 平均分配、按比例或手动输入金额
- **项目管理**: 按项目整理支出
- **收据附件**: 添加收据图片作为参考
- **结算追踪**: 将单个支出或整个项目标记为已结算
- **离线支持**: 离线工作并在连接时同步（Pro）

## 开始使用

1. 从 App Store 下载 Evere（iOS）
2. 创建账户或使用访客模式
3. 创建项目并邀请参与者
4. 添加支出并分账
5. 追踪结算
