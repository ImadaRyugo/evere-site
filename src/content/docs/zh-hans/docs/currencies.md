Evere 支持 **162 种货币**。无论海外旅行还是多币种混用，都能原样记录并结算。

## 基准货币与可用货币

项目里有两种关于货币的概念。

- **基准货币** — 用于结算计算的货币。所有支出都会换算成它来汇总。创建项目时选择，且首笔支出后无法更改。
- **可用货币** — 记录支出时可以选择的货币。可随时在项目设置中添加或删除。

例如，把基准货币设为“CNY（人民币）”，并把“THB（泰铢）”加入可用货币，那么用泰铢付的支出也会自动换算成人民币来结算。

## 汇率的工作方式

记录支出时，**会自动获取并保存当时的汇率。**

:::note
汇率会固定为记账当时的值。之后行情波动，也不会改变已记录支出的换算金额。
:::

在支出详情页和结算界面，可查看所用汇率及其最后更新时间。

## 注意未确认的汇率

如果某笔支出因故汇率未确认，**该笔支出会被暂时排除在结算计算之外。** 若结算界面提示“存在未确认的汇率”，请打开对应支出并确认汇率。若一直被排除，结算金额就会与实际不符。

## 手动设置汇率（Pro）

若想自己设定汇率，可为每笔支出输入**手动汇率**。

:::note
手动汇率输入是 Pro 方案的功能。
:::

## 我的货币（用于显示的货币）

在设置标签的“我的货币”中，可设置**你自己的显示货币**。对于与之不同的金额，会一并显示换算金额。

:::note
我的货币仅用于显示，**不影响结算计算结果。**
:::
