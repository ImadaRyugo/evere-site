## 免费方案与 Pro 方案

Evere 免费，分账所需的功能都能用。想解除上限或让使用更方便时，选择 Pro。

| 功能 | 免费 | Pro |
| --- | --- | --- |
| 项目数量 | 最多 3 个 | 无限 |
| 收据图片 | 每项目 5 张 | 无限 |
| 多币种·自动换算 | ✓ | ✓ |
| 邀请成员·实时同步 | ✓ | ✓ |
| 手动汇率 | — | ✓ |
| 预算设置·达标通知 | — | ✓ |
| 离线 | — | ✓ |
| 广告 | 显示 | 隐藏 |

:::note
价格和购买方案（月付、年付、买断）请在应用内查看。Pro 可能提供免费试用。
:::

### 升级与恢复购买

- 在设置标签的“账号管理”或“方案详情”中升级。
- 换手机等情况下要沿用购买时，使用“**恢复购买**”。
- 订阅通过 Apple ID 计费，可随时在 Apple 的订阅设置中管理或取消。

## 常见问题

### 结算金额对不上 / 某人没显示

带有**未确认汇率**的支出会被排除在结算计算之外。请查看结算界面是否有提示，并确认对应支出的汇率（[多币种与汇率](/zh-hans/docs/currencies/#注意未确认的汇率)）。

### 无法添加支出

该项目可能已被标记为**已结算**。在项目列表中取消结算即可再次添加。

### 无法更改基准货币

记录第一笔支出后，基准货币就无法更改。若想使用其他基准货币，请创建新项目（[项目](/zh-hans/docs/projects/#货币设置)）。

### 无法邀请或记录结算

这些操作需要注册账号，访客模式无法使用。请在设置标签中注册账号（[账号与数据](/zh-hans/docs/account/)）。

### 换手机后数据丢失 / 快要丢失

访客模式的数据只保存在设备上。换手机前，请务必注册账号，把数据保存到云端。

### 想创建 4 个以上项目 / 想去掉广告

这些都能用 Pro 方案解决（见上方对比表）。

### 仍未解决时

请通过[联系表单](/zh-hans/support/)与我们联系。也可从应用设置标签的“支持”中联系我们。
