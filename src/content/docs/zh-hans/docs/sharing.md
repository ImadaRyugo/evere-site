Evere 既能和有应用的人一起用，也能和没应用的人一起用。添加参与者有两种方式。

## 用名字添加参与者

最简单的方式就是**用名字添加参与者**。在创建项目时，或在项目设置的“参与者管理”中输入名字即可添加。

- 即使对方没有应用也能添加。
- 添加的参与者可作为分摊对象来选择。
- 结算结果可通过[分享报告](/zh-hans/docs/settlement/#分享结算报告)传给对方。

一起用的人不必都安装应用。无论是情侣，还是不用应用的朋友，用这种方式都能搞定。

## 用邀请链接邀请成员

如果对方也用 Evere，可以用**邀请链接**邀请其加入项目。被邀请的成员可在自己的设备上实时共享同一个项目。

- 在项目详情中创建邀请链接，以链接或二维码分享。
- 邀请链接可以停用，也可能会过期。

:::note
创建邀请链接需要注册账号，访客模式无法创建。此外，通过链接**加入的一方**也需要注册账号。
:::

## 该用哪种？

- 对方不用应用，或想快速搞定 → **用名字添加**
- 对方也想一起实时记账、查看 → **邀请链接**

在同一个项目里也可以两种方式混用。
