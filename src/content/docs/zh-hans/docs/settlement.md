随着支出不断记录，“结算”标签会自动显示“谁该给谁多少”。繁琐的分账计算交给 Evere 就好。

## 如何看结算界面

结算标签会显示以下信息。

- **摘要** — 总支出数和总额
- **结算状况** — 已结算与未结算各自的金额与比例
- **支出对比** — 用图表对比每位参与者的“支付额（实际付出）”和“负担额（本应承担）”
- **按货币 / 按类别** — 按货币或类别的明细
- **结算** — 谁该给谁多少的列表
- **预算** — 若设置了预算，则显示使用情况（[预算是 Pro 功能](/zh-hans/docs/features/#预算pro)）

## 智能结算与普通结算

结算的显示方式可从两种中选择（在设置中切换）。

- **智能结算** — 汇总借贷关系，按**所需结算次数最少**的方式显示。来回更少即可结清。
- **普通结算** — 按谁该给谁多少，逐对原样显示。

## 标记为已结算

实际把钱付清后，在结算界面点击该条目并确认，标记为**已结算**。

- 标记为已结算的金额会从之后的计算中排除。
- 状态有“已结算”“部分结算”“未结算”。
- 弄错时可**取消结算**来还原。

结算可按三种粒度管理。

- **单笔支出** — 把某笔支出标记为已结算
- **单条结算流** — 在结算界面按“谁给谁”的条目逐条标记为已结算
- **整个项目** — 一次性标记为已结算（见[项目](/zh-hans/docs/projects/#将项目标记为已结算)）

:::note
记录结算（标记为已结算的操作）需要注册账号，访客模式无法使用。详见[账号与数据](/zh-hans/docs/account/)。
:::

## 分享结算报告

在结算界面选择“分享结算报告”，即可把结算结果作为排好版的文字分享出去。粘贴到群聊里，没有应用的人也能看到结果。

分享时可选择：

- **分享内容** — 仅结算结果 / 含支出历史
- 是否包含参与者信息和收据图片
- 是否附上以辅助货币换算的金额
