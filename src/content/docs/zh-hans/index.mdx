## 关于 Evere

Evere 是一款帮助情侣和团体轻松管理分账和结算的移动应用程序。支持多种货币。

<CardGrid>
	<Card title="多货币支持" icon="translate">
		以不同货币记录支出并自动换算
	</Card>
	<Card title="灵活分账" icon="puzzle">
		平均分配、按比例或手动输入金额
	</Card>
	<Card title="项目管理" icon="list-format">
		按项目整理支出
	</Card>
	<Card title="结算追踪" icon="approve-check">
		将单个支出或整个项目标记为已结算
	</Card>
</CardGrid>
