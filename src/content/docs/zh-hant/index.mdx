## 關於 Evere

Evere 是一款幫助情侶和團體輕鬆管理分帳和結算的行動應用程式。支援多種貨幣。

<CardGrid stagger>
	<Card title="多貨幣支援" icon="translate">
		以不同貨幣記錄支出並自動換算
	</Card>
	<Card title="靈活分帳" icon="puzzle">
		平均分配、按比例或手動輸入金額
	</Card>
	<Card title="專案管理" icon="suitcase">
		按專案整理支出
	</Card>
	<Card title="結算追蹤" icon="approve-check">
		將單個支出或整個專案標記為已結算
	</Card>
</CardGrid>

## 支援平台

- iOS 15.1 及更高版本
- Android 7.0 及更高版本
