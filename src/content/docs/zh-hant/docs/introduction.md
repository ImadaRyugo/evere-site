# 歡迎使用 Evere

Evere 是一款幫助情侶和團體輕鬆管理分帳和結算的行動應用程式。支援多種貨幣。

## 主要功能

- **多貨幣支援**: 以不同貨幣記錄支出並自動換算
- **靈活分帳**: 平均分配、按比例或手動輸入金額
- **專案管理**: 按專案整理支出
- **收據附件**: 新增收據圖片作為參考
- **結算追蹤**: 將單個支出或整個專案標記為已結算
- **離線支援**: 離線工作並在連線時同步

## 開始使用

1. 從 App Store (iOS) 或 Google Play (Android) 下載 Evere
2. 建立帳戶或使用訪客模式
3. 建立專案並邀請參與者
4. 新增支出並分帳
5. 追蹤結算
