Evere 是一款讓情侶與團體輕鬆分帳、結算的 iOS App。支援多種貨幣，從旅行到日常開銷，一個 App 就搞定。

跟著這份指南，你就能把 Evere 的所有功能運用自如。先來認識核心概念與畫面結構。

## Evere 的 4 個核心概念

記住這四點，其餘的就容易理解了。

- **專案** — 分帳的「單位」。依情境各建一個，例如旅行、約會、同居等。所有支出都記錄在專案裡。
- **基準貨幣** — 決定該專案用哪種貨幣結算的貨幣。建立專案時選擇。
- **參與者** — 一起使用專案的成員。有 App 的人可透過邀請連結加入，沒有 App 的人只填名字也能新增。
- **結算** — Evere 自動算出「誰該給誰多少」。只要記帳，隨時都能查看結算狀況。

## 畫面結構（5 個分頁）

Evere 底部有 5 個分頁。

| 分頁 | 作用 |
| --- | --- |
| **專案** | 專案的列表、建立、切換與設定 |
| **支出** | 目前專案的支出列表 |
| **新增** | 記錄支出 |
| **結算** | 「誰給誰多少」的計算結果與結算狀況 |
| **設定** | 帳號、方案、貨幣、通知等設定 |

支出與結算分頁會顯示**目前使用中的專案**。在專案分頁選擇其他專案即可切換所顯示的內容。

## 支援環境

- 支援 **iOS 15.1 以上**的 iPhone。
- 目前尚無 Android 版。
