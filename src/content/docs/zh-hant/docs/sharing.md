Evere 既能和有 App 的人一起用，也能和沒 App 的人一起用。新增參與者有兩種方式。

## 用名字新增參與者

最簡單的方式就是**用名字新增參與者**。在建立專案時，或在專案設定的「參與者管理」中輸入名字即可新增。

- 即使對方沒有 App 也能新增。
- 新增的參與者可作為分攤對象來選擇。
- 結算結果可透過[分享報告](/zh-hant/docs/settlement/#分享結算報告)傳給對方。

一起用的人不必都安裝 App。無論是情侶，還是不用 App 的朋友，用這種方式都能搞定。

## 用邀請連結邀請成員

如果對方也用 Evere，可以用**邀請連結**邀請其加入專案。被邀請的成員可在自己的裝置上即時共享同一個專案。

- 在專案詳情中建立邀請連結，以連結或 QR code 分享。
- 邀請連結可以停用，也可能會過期。

:::note
建立邀請連結需要註冊帳號，訪客模式無法建立。此外，透過連結**加入的一方**也需要註冊帳號。
:::

## 該用哪一種？

- 對方不用 App，或想快速搞定 → **用名字新增**
- 對方也想一起即時記帳、查看 → **邀請連結**

在同一個專案裡也可以兩種方式混用。
