專案是 Evere 的核心。依情境各建一個，例如旅行、約會、同居，並在其中記錄支出。

## 建立專案

在「專案」分頁的「建立新專案」中建立。

- **專案名稱**（20 字以內）
- **要用的貨幣** — 可選多個。旅行時把當地貨幣和自己的貨幣都選上會很方便。
- **基準貨幣** — 所選貨幣中的一個，結算會用它計算。系統會依裝置的語言與地區自動推薦。
- **參與者**（選填） — 用名字新增，之後也可以新增。

:::note
建立專案僅在連線時可用。
:::

:::caution
**免費方案最多可建立 3 個專案。** 若想建立更多，升級到 Pro 即可無限建立。
:::

## 使用中的專案

在列表中選擇某個專案，它就成為**使用中的專案**。「支出」與「結算」分頁會顯示這個使用中的專案。要處理其他專案，請在列表中重新選擇。

## 修改專案設定

在專案設定中可以修改以下項目（多數操作僅限專案**擁有者**）。

### 專案名稱

可隨時修改（20 字以內）。

### 貨幣設定

- **可用貨幣** — 可隨時新增或刪除。
- **基準貨幣** — 結算計算的基準。

:::caution
**記錄第一筆支出後，基準貨幣就無法變更。** 這是為了確保結算的一致性。之後只能新增或刪除可用貨幣。若想使用其他基準貨幣，請建立新專案。
:::

### 管理參與者

可新增參與者、修改其名字或刪除。記錄支出時可選擇參與者。

:::note
已包含在某筆支出中的參與者無法刪除。請先將其從該支出中移除，或編輯／刪除該支出。
:::

## 將專案標記為已結算

當一切都結清後，可以把整個專案標記為**已結算**。之後就無法再向該專案新增支出。

若想再次新增支出，**取消結算**即可重新編輯。

## 刪除專案

不再需要的專案可以刪除。

:::caution
僅擁有者可刪除，且僅在連線時可用。**刪除後無法復原。**
:::
