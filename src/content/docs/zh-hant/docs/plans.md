## 免費方案與 Pro 方案

Evere 免費，分帳所需的功能都能用。想解除上限或讓使用更方便時，選擇 Pro。

| 功能 | 免費 | Pro |
| --- | --- | --- |
| 專案數量 | 最多 3 個 | 無限 |
| 收據圖片 | 每專案 5 張 | 無限 |
| 多幣別·自動換算 | ✓ | ✓ |
| 邀請成員·即時同步 | ✓ | ✓ |
| 手動匯率 | — | ✓ |
| 預算設定·達標通知 | — | ✓ |
| 離線 | — | ✓ |
| 廣告 | 顯示 | 隱藏 |

:::note
價格與購買方案（月付、年付、買斷）請在 App 內查看。Pro 可能提供免費試用。
:::

### 升級與還原購買

- 在設定分頁的「帳號管理」或「方案詳情」中升級。
- 換手機等情況下要沿用購買時，使用「**還原購買**」。
- 訂閱透過 Apple ID 收費，可隨時在 Apple 的訂閱設定中管理或取消。

## 常見問題

### 結算金額對不上 / 某人沒顯示

帶有**未確認匯率**的支出會被排除在結算計算之外。請查看結算畫面是否有提示，並確認對應支出的匯率（[多幣別與匯率](/zh-hant/docs/currencies/#注意未確認的匯率)）。

### 無法新增支出

該專案可能已被標記為**已結算**。在專案列表中取消結算即可再次新增。

### 無法變更基準貨幣

記錄第一筆支出後，基準貨幣就無法變更。若想使用其他基準貨幣，請建立新專案（[專案](/zh-hant/docs/projects/#貨幣設定)）。

### 無法邀請或記錄結算

這些操作需要註冊帳號，訪客模式無法使用。請在設定分頁中註冊帳號（[帳號與資料](/zh-hant/docs/account/)）。

### 換手機後資料遺失 / 快要遺失

訪客模式的資料只儲存在裝置上。換手機前，請務必註冊帳號，把資料儲存到雲端。

### 想建立 4 個以上專案 / 想去掉廣告

這些都能用 Pro 方案解決（見上方對比表）。

### 仍未解決時

請透過[聯絡表單](/zh-hant/support/)與我們聯絡。也可從 App 設定分頁的「支援」中聯絡我們。
