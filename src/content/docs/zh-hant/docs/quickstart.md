Evere 的基本流程只有「**建立專案 → 記錄支出 → 結算**」。這裡以旅行為例走一遍，大約 5 分鐘。

## 第 1 步：開始使用 Evere

從 App Store 下載 Evere 並開啟，然後選擇開始方式。

- **用 Apple / Google / 電子郵件註冊** — 資料會儲存到雲端，邀請、記錄結算等所有功能都能用。
- **以訪客身分開始** — 免註冊即可立即使用，但部分功能會受限（詳見[帳號與資料](/zh-hant/docs/account/)）。

先設定**顯示名稱**。這個名字會顯示給專案成員（之後可以修改）。

## 第 2 步：建立專案

在「專案」分頁建立新專案。

1. 輸入**專案名稱**（例如「日本旅行」）。
2. 選擇**要用的貨幣**。海外旅行時，把當地貨幣（如 JPY）和自己的貨幣（如 TWD）都選上會很方便。
3. 所選貨幣中的一個會成為**基準貨幣**，結算會用這個貨幣計算。
4. 如有需要，用名字新增**參與者**（之後也可以新增）。

:::note
記錄第一筆支出後，基準貨幣就無法變更了。它用於所有結算計算，因此建立專案時請仔細確認。
:::

## 第 3 步：記錄支出

在「新增」分頁記錄支出。

1. 輸入**金額**和**貨幣**。
2. 選擇**付款人**（先墊付的人）。
3. 選擇**參與者**（分攤這筆支出的人）和**分攤方式**（平均分攤、按比例或按金額）。
4. 視需要新增類別、標題、日期、備註和收據圖片。

用「繼續新增」可以連續記錄多筆。

## 第 4 步：查看結算狀況

開啟「結算」分頁，就能看到 Evere 自動算出的結果。

- **摘要**：總支出數和總額
- **結算狀況**：已結算與未結算的比例
- **支出對比**：每位參與者「付了多少（支付額）」與「該分擔多少（負擔額）」
- **結算**：誰該給誰多少

## 第 5 步：結算並記錄

實際把錢付清後，在結算畫面點按該項目，標記為**已結算**。已結算的部分會從之後的計算中排除，並記錄誰已經結清。

到這裡基本流程就走完了。接下來的頁面會更詳細地介紹各項功能。
