Evere 支援 **162 種貨幣**。無論海外旅行還是多幣別混用，都能原樣記錄並結算。

## 基準貨幣與可用貨幣

專案裡有兩種關於貨幣的概念。

- **基準貨幣** — 用於結算計算的貨幣。所有支出都會換算成它來彙總。建立專案時選擇，且首筆支出後無法變更。
- **可用貨幣** — 記錄支出時可以選擇的貨幣。可隨時在專案設定中新增或刪除。

例如，把基準貨幣設為「TWD（新台幣）」，並把「JPY（日圓）」加入可用貨幣，那麼用日圓付的支出也會自動換算成新台幣來結算。

## 匯率的運作方式

記錄支出時，**會自動取得並儲存當時的匯率。**

:::note
匯率會固定為記帳當時的值。之後行情波動，也不會改變已記錄支出的換算金額。
:::

在支出詳情頁和結算畫面，可查看所用匯率及其最後更新時間。

## 注意未確認的匯率

如果某筆支出因故匯率未確認，**該筆支出會被暫時排除在結算計算之外。** 若結算畫面提示「有未確認的匯率」，請開啟對應支出並確認匯率。若一直被排除，結算金額就會與實際不符。

## 手動設定匯率（Pro）

若想自己設定匯率，可為每筆支出輸入**手動匯率**。

:::note
手動匯率輸入是 Pro 方案的功能。
:::

## 我的貨幣（用於顯示的貨幣）

在設定分頁的「我的貨幣」中，可設定**你自己的顯示貨幣**。對於與之不同的金額，會一併顯示換算金額。

:::note
我的貨幣僅用於顯示，**不影響結算計算結果。**
:::
