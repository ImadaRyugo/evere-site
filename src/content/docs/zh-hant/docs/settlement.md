隨著支出不斷記錄，「結算」分頁會自動顯示「誰該給誰多少」。繁瑣的分帳計算交給 Evere 就好。

## 如何看結算畫面

結算分頁會顯示以下資訊。

- **摘要** — 總支出數和總額
- **結算狀況** — 已結算與未結算各自的金額與比例
- **支出對比** — 用圖表對比每位參與者的「支付額（實際付出）」和「負擔額（本應承擔）」
- **依貨幣 / 依類別** — 依貨幣或類別的明細
- **結算** — 誰該給誰多少的列表
- **預算** — 若設定了預算，則顯示使用情況（[預算是 Pro 功能](/zh-hant/docs/features/#預算pro)）

## 智慧結算與一般結算

結算的顯示方式可從兩種中選擇（在設定中切換）。

- **智慧結算** — 彙整借貸關係，依**所需結算次數最少**的方式顯示。來回更少即可結清。
- **一般結算** — 依誰該給誰多少，逐對原樣顯示。

## 標記為已結算

實際把錢付清後，在結算畫面點按該項目並確認，標記為**已結算**。

- 標記為已結算的金額會從之後的計算中排除。
- 狀態有「已結算」「部分結算」「未結算」。
- 弄錯時可**取消結算**來還原。

結算可依三種層級管理。

- **單筆支出** — 把某筆支出標記為已結算
- **單條結算流** — 在結算畫面依「誰給誰」的項目逐條標記為已結算
- **整個專案** — 一次標記為已結算（見[專案](/zh-hant/docs/projects/#將專案標記為已結算)）

:::note
記錄結算（標記為已結算的操作）需要註冊帳號，訪客模式無法使用。詳見[帳號與資料](/zh-hant/docs/account/)。
:::

## 分享結算報告

在結算畫面選擇「分享結算報告」，即可把結算結果作為排好版的文字分享出去。貼到群組聊天裡，沒有 App 的人也能看到結果。

分享時可選擇：

- **分享內容** — 僅結算結果 / 含支出紀錄
- 是否包含參與者資訊和收據圖片
- 是否附上以輔助貨幣換算的金額
