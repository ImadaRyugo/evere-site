除了基本的分帳，Evere 還有讓日常更順手的功能。

## 類別

用「餐飲」「交通」等類別為支出分類。系統預設了若干類別，也可以**自訂新增類別**。

- 在設定分頁的「類別管理」中，用表情符號和名稱（20 字以內）新增、編輯或刪除類別。
- 類別**在所有專案間通用**。
- 在結算畫面可查看依類別的明細。

## 預算（Pro）

為專案設定**預算**，當使用量達到你設定的水準時收到通知。

- 設定預算金額（用**基準貨幣**設定）。
- 作為**達標通知**，可新增依百分比（如 80%）或金額（如 50000）的多個條件。達到設定水準時會收到通知。
- 在結算畫面可查看所設預算和已用情況。

:::note
僅擁有者可設定預算。此外，預算設定是 Pro 方案的功能。
:::

## 通知

透過推播通知了解成員的操作與專案狀況。可在設定分頁的「通知」中開關。

- **支出通知** — 支出的新增、更新、刪除
- **結算通知** — 單筆支出、整個專案、最終結算的結算/取消
- **預算提醒**

:::note
要使用推播通知，請在 iPhone 設定中允許 Evere 的通知。
:::

## 離線（Pro）

在沒有訊號的地方也能記錄支出，回到連線時會**自動同步**。旅途中訊號不穩時也讓人安心。

- 在設定分頁的「離線同步」中，可查看連線狀態和待處理的變更，也可手動同步。

:::note
離線使用是 Pro 方案的功能。另外，建立/刪除專案、建立邀請連結、變更結算狀態、管理方案等部分操作僅在連線時可用。
:::
