Evere ใช้ร่วมกับคนที่มีแอปและคนที่ไม่มีแอปก็ได้ วิธีเพิ่มผู้เข้าร่วมมีสองแบบ

## เพิ่มผู้เข้าร่วมด้วยชื่อ

วิธีที่ง่ายที่สุดคือ **เพิ่มผู้เข้าร่วมด้วยชื่อ** ใส่ชื่อตอนสร้างโปรเจกต์ หรือจาก "จัดการผู้เข้าร่วม" ในการตั้งค่าโปรเจกต์ ก็เพิ่มได้เลย

- เพิ่มได้แม้อีกฝ่ายไม่มีแอป
- ผู้เข้าร่วมที่เพิ่มไว้เลือกเป็นคนหารค่าใช้จ่ายได้
- ส่งผลการชำระให้เขาได้ผ่าน [การแชร์รายงาน](/th/docs/settlement/#แชร์รายงานการชำระ)

ไม่จำเป็นที่ทุกคนต้องติดตั้งแอป ไม่ว่าจะเป็นคู่รักหรือเพื่อนที่ไม่ใช้แอป ก็จบด้วยวิธีนี้ได้

## เชิญสมาชิกด้วยลิงก์เชิญ

ถ้าอีกฝ่ายก็ใช้ Evere เชิญเข้าโปรเจกต์ด้วย **ลิงก์เชิญ** ได้ สมาชิกที่ถูกเชิญจะแชร์โปรเจกต์เดียวกันแบบเรียลไทม์บนเครื่องของตัวเองได้

- สร้างลิงก์เชิญจากหน้ารายละเอียดโปรเจกต์ แล้วแชร์เป็นลิงก์หรือ QR code
- ลิงก์เชิญปิดการใช้งานได้ และอาจหมดอายุได้

:::note
การสร้างลิงก์เชิญต้องสมัครบัญชี ทำในโหมดแขกไม่ได้ และคนที่ **เข้าร่วม** ผ่านลิงก์ก็ต้องสมัครบัญชีเช่นกัน
:::

## ควรใช้แบบไหนดี?

- อีกฝ่ายไม่ใช้แอป หรืออยากให้เร็ว → **เพิ่มด้วยชื่อ**
- อีกฝ่ายอยากบันทึกและเช็กแบบเรียลไทม์ด้วย → **ลิงก์เชิญ**

ในโปรเจกต์เดียวจะผสมทั้งสองแบบก็ได้
