## แผน Free กับแผน Pro

Evere ใช้ฟรี และมีฟีเจอร์ที่จำเป็นสำหรับการหารบิลครบถ้วน เมื่ออยากปลดข้อจำกัดหรือเพิ่มความสะดวก ลองใช้ Pro

| ฟีเจอร์ | Free | Pro |
| --- | --- | --- |
| จำนวนโปรเจกต์ | สูงสุด 3 | ไม่จำกัด |
| รูปใบเสร็จ | 5 รูปต่อโปรเจกต์ | ไม่จำกัด |
| หลายสกุลเงิน·แปลงอัตโนมัติ | ✓ | ✓ |
| เชิญสมาชิก·ซิงก์เรียลไทม์ | ✓ | ✓ |
| อัตราแลกเปลี่ยนด้วยตนเอง | — | ✓ |
| ตั้งงบประมาณ·แจ้งเตือนเมื่อถึงเกณฑ์ | — | ✓ |
| ออฟไลน์ | — | ✓ |
| โฆษณา | แสดง | ซ่อน |

:::note
ราคาและแผนการซื้อ (รายเดือน รายปี ซื้อขาด) ดูได้ในแอป Pro อาจมีทดลองใช้ฟรี
:::

### อัปเกรดและกู้คืนการซื้อ

- อัปเกรดได้จาก "จัดการบัญชี" หรือ "รายละเอียดแผน" ในแท็บตั้งค่า
- เมื่อเปลี่ยนเครื่องแล้วอยากโอนการซื้อมา ใช้ "**กู้คืนการซื้อ**"
- การสมัครสมาชิกเรียกเก็บผ่าน Apple ID และจัดการหรือยกเลิกได้ทุกเมื่อจากการตั้งค่าการสมัครสมาชิกของ Apple

## การแก้ปัญหา

### ยอดการชำระไม่ตรง หรือไม่มีบางคนแสดง

ค่าใช้จ่ายที่มี **อัตราแลกเปลี่ยนที่ยังไม่ยืนยัน** จะถูกตัดออกจากการคำนวณการชำระ ตรวจว่าหน้าชำระบัญชีมีคำเตือนหรือไม่ แล้วยืนยันอัตราของรายการนั้น ([หลายสกุลเงินและอัตราแลกเปลี่ยน](/th/docs/currencies/#ระวังอัตราที่ยังไม่ยืนยัน))

### เพิ่มค่าใช้จ่ายไม่ได้

โปรเจกต์นั้นอาจอยู่ในสถานะ **ชำระแล้ว** ยกเลิกการชำระจากรายการโปรเจกต์ แล้วจะเพิ่มได้อีก

### เปลี่ยนสกุลเงินหลักไม่ได้

เมื่อบันทึกค่าใช้จ่ายรายการแรกแล้ว จะเปลี่ยนสกุลเงินหลักไม่ได้ หากต้องการสกุลเงินหลักอื่น ให้สร้างโปรเจกต์ใหม่ ([โปรเจกต์](/th/docs/projects/#การตั้งค่าสกุลเงิน))

### เชิญหรือบันทึกการชำระไม่ได้

สิ่งเหล่านี้ต้องสมัครบัญชี ทำในโหมดแขกไม่ได้ สมัครบัญชีจากแท็บตั้งค่า ([บัญชีและข้อมูล](/th/docs/account/))

### ข้อมูลหายเพราะเปลี่ยนเครื่อง หรือกำลังจะหาย

ข้อมูลในโหมดแขกถูกเก็บไว้เฉพาะในเครื่อง ก่อนเปลี่ยนเครื่อง อย่าลืมสมัครบัญชีเพื่อเก็บข้อมูลขึ้นคลาวด์

### อยากสร้างโปรเจกต์เกิน 3 อัน หรืออยากซ่อนโฆษณา

แก้ได้ด้วยแผน Pro (ดูตารางเปรียบเทียบด้านบน)

### ยังแก้ไม่ได้

ติดต่อผ่าน [แบบฟอร์มติดต่อ](/th/support/) และติดต่อได้จาก "ซัพพอร์ต" ในแท็บตั้งค่าของแอปด้วย
