เมื่อบันทึกค่าใช้จ่ายไปเรื่อย ๆ แท็บ "ชำระบัญชี" จะแสดง "ใครต้องจ่ายให้ใครเท่าไหร่" ให้อัตโนมัติ ปล่อยการคำนวณหารบิลที่ยุ่งยากให้ Evere จัดการ

## วิธีอ่านหน้าชำระบัญชี

แท็บชำระบัญชีแสดงข้อมูลต่อไปนี้

- **สรุป** — จำนวนรายการจ่ายและยอดรวม
- **สถานะการชำระ** — ยอดและสัดส่วนของที่ชำระแล้วกับยังไม่ชำระ
- **การเปรียบเทียบการจ่าย** — เปรียบเทียบ "ยอดที่จ่ายไป" กับ "ยอดที่ต้องรับผิดชอบ" ของแต่ละคนด้วยกราฟ
- **ตามสกุลเงิน / ตามหมวดหมู่** — แยกยอดตามสกุลเงินหรือหมวดหมู่
- **การชำระ** — รายการว่าใครต้องจ่ายให้ใครเท่าไหร่
- **งบประมาณ** — สถานะการใช้ หากตั้งงบไว้ ([งบประมาณเป็นฟีเจอร์ Pro](/th/docs/features/#งบประมาณ-pro))

## ชำระอัจฉริยะกับชำระปกติ

เลือกวิธีแสดงการชำระได้สองแบบ (สลับในการตั้งค่า)

- **ชำระอัจฉริยะ** — รวบยอดที่ยืมกันไว้ แล้วแสดงแบบที่ **ใช้จำนวนครั้งการชำระน้อยที่สุด** โอนไปมาน้อยครั้งก็จบ
- **ชำระปกติ** — แสดงว่าใครต้องจ่ายให้ใครเท่าไหร่ เป็นรายคู่ตามนั้น

## ทำเครื่องหมายว่าชำระแล้ว

เมื่อโอนเงินกันจริงเรียบร้อยแล้ว แตะรายการในหน้าชำระบัญชี ยืนยัน แล้วทำเครื่องหมายว่า **ชำระแล้ว**

- ยอดที่ชำระแล้วจะถูกตัดออกจากการคำนวณต่อไป
- สถานะมี "ชำระแล้ว" "ชำระบางส่วน" และ "ยังไม่ชำระ"
- ถ้าทำผิด **ยกเลิกการชำระ** เพื่อย้อนกลับได้

จัดการการชำระได้ 3 ระดับ

- **รายรายการ** — ทำเครื่องหมายค่าใช้จ่ายแต่ละรายการว่าชำระแล้ว
- **ตามรายการชำระ** — ทำเครื่องหมายตามรายการ "ใครจ่ายให้ใคร" ในหน้าชำระบัญชี
- **ทั้งโปรเจกต์** — ทำเครื่องหมายว่าชำระแล้วทีเดียว (ดู [โปรเจกต์](/th/docs/projects/#ทำเครื่องหมายว่าโปรเจกต์ชำระแล้ว))

:::note
การบันทึกการชำระ (ทำเครื่องหมายว่าชำระแล้ว) ต้องสมัครบัญชี ทำในโหมดแขกไม่ได้ ดู [บัญชีและข้อมูล](/th/docs/account/)
:::

## แชร์รายงานการชำระ

เลือก "แชร์รายงานชำระ" ในหน้าชำระบัญชี เพื่อแชร์ผลการชำระเป็นข้อความที่จัดรูปแบบไว้ วางในแชทกลุ่มได้ แม้คนที่ไม่มีแอปก็เห็นผลลัพธ์ได้

ตอนแชร์เลือกได้ว่า

- **เนื้อหาที่แชร์** — เฉพาะผลการชำระ / รวมประวัติ
- จะใส่ข้อมูลผู้เข้าร่วมและรูปใบเสร็จหรือไม่
- จะแนบยอดแปลงในสกุลเงินรองหรือไม่
