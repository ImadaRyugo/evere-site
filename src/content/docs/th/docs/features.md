นอกจากการหารบิลพื้นฐานแล้ว Evere ยังมีฟีเจอร์ที่ช่วยให้ใช้งานในแต่ละวันสะดวกขึ้น

## หมวดหมู่

จัดกลุ่มค่าใช้จ่ายด้วยหมวดหมู่ เช่น "ค่าอาหาร" หรือ "ค่าเดินทาง" มีหมวดหมู่พื้นฐานให้อยู่แล้ว และ **เพิ่มหมวดหมู่เองได้** ด้วย

- จาก "จัดการหมวดหมู่" ในแท็บตั้งค่า เพิ่ม แก้ไข หรือลบหมวดหมู่ด้วยอีโมจิและชื่อ (ไม่เกิน 20 ตัวอักษร) ได้
- หมวดหมู่ **ใช้ร่วมกันในทุกโปรเจกต์**
- ในหน้าชำระบัญชี ดูยอดแยกตามหมวดหมู่ได้

## งบประมาณ (Pro)

ตั้ง **งบประมาณ** ให้โปรเจกต์ และรับการแจ้งเตือนเมื่อการใช้จ่ายถึงระดับที่กำหนด

- ตั้งจำนวนงบประมาณ (ตั้งด้วย **สกุลเงินหลัก**)
- ตั้ง **การแจ้งเตือนเมื่อถึงเกณฑ์** เป็นเปอร์เซ็นต์ (เช่น 80%) หรือจำนวนเงิน (เช่น 50,000) หลายเงื่อนไขได้ เมื่อถึงระดับที่ตั้งไว้จะมีแจ้งเตือน
- ในหน้าชำระบัญชี ดูงบที่ตั้งไว้และยอดที่ใช้ไปได้

:::note
เฉพาะเจ้าของเท่านั้นที่ตั้งงบประมาณได้ และการตั้งงบเป็นฟีเจอร์ของแผน Pro
:::

## การแจ้งเตือน

รับการแจ้งเตือนแบบ Push เกี่ยวกับการกระทำของสมาชิกและสถานะโปรเจกต์ เปิด/ปิดได้จาก "การแจ้งเตือน" ในแท็บตั้งค่า

- **แจ้งเตือนค่าใช้จ่าย** — เพิ่ม แก้ไข หรือลบค่าใช้จ่าย
- **แจ้งเตือนการชำระ** — ชำระ/ยกเลิกในระดับรายการ ทั้งโปรเจกต์ หรือการชำระขั้นสุดท้าย
- **แจ้งเตือนงบประมาณ**

:::note
หากต้องการใช้การแจ้งเตือนแบบ Push ให้อนุญาตการแจ้งเตือนของ Evere ในการตั้งค่า iPhone
:::

## ออฟไลน์ (Pro)

บันทึกค่าใช้จ่ายได้แม้ไม่มีสัญญาณ และจะ **ซิงก์อัตโนมัติ** เมื่อกลับมาออนไลน์ อุ่นใจเมื่อสัญญาณไม่เสถียรระหว่างทริป

- จาก "ซิงก์ออฟไลน์" ในแท็บตั้งค่า ดูสถานะออนไลน์และการเปลี่ยนแปลงที่ค้างอยู่ และซิงก์เองได้

:::note
การใช้ออฟไลน์เป็นฟีเจอร์ของแผน Pro อนึ่ง บางอย่าง เช่น การสร้าง/ลบโปรเจกต์ การสร้างลิงก์เชิญ การเปลี่ยนสถานะการชำระ และการจัดการแผน ทำได้เฉพาะตอนออนไลน์
:::
