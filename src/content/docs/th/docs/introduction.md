# ยินดีต้อนรับสู่ Evere

Evere เป็นแอปพลิเคชันสำหรับจัดการการแบ่งบิลและการชำระเงินระหว่างคู่รักและกลุ่ม รองรับหลายสกุลเงิน

## คุณสมบัติหลัก

- **รองรับหลายสกุลเงิน**: จัดการค่าใช้จ่ายในสกุลเงินต่างๆ พร้อมแปลงอัตโนมัติ
- **การแบ่งแบบยืดหยุ่น**: แบ่งเท่าๆ กัน, แบ่งตามอัตราส่วน, หรือระบุจำนวนเอง
- **การจัดการโครงการ**: จัดระเบียบค่าใช้จ่ายตามโครงการ
- **แนบใบเสร็จ**: เพิ่มรูปภาพใบเสร็จเพื่ออ้างอิง
- **ติดตามการชำระเงิน**: ทำเครื่องหมายค่าใช้จ่ายหรือโครงการเป็นชำระแล้ว
- **รองรับออฟไลน์**: ทำงานแบบออฟไลน์และซิงค์เมื่อเชื่อมต่อ

## เริ่มต้นใช้งาน

1. ดาวน์โหลด Evere จาก App Store (iOS) หรือ Google Play (Android)
2. สร้างบัญชีหรือใช้โหมดผู้เยี่ยมชม
3. สร้างโครงการและเชิญผู้เข้าร่วม
4. เพิ่มค่าใช้จ่ายและแบ่งบิล
5. ติดตามการชำระเงิน
