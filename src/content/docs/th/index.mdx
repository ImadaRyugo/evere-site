## เกี่ยวกับ Evere

Evere เป็นแอปพลิเคชันสำหรับจัดการการแบ่งบิลและการชำระเงินระหว่างคู่รักและกลุ่ม รองรับหลายสกุลเงิน

<CardGrid stagger>
	<Card title="รองรับหลายสกุลเงิน" icon="translate">
		จัดการค่าใช้จ่ายในสกุลเงินต่างๆ พร้อมแปลงอัตโนมัติ
	</Card>
	<Card title="การแบ่งแบบยืดหยุ่น" icon="puzzle">
		แบ่งเท่าๆ กัน, แบ่งตามอัตราส่วน, หรือระบุจำนวนเอง
	</Card>
	<Card title="การจัดการโครงการ" icon="suitcase">
		จัดระเบียบค่าใช้จ่ายตามโครงการ
	</Card>
	<Card title="ติดตามการชำระเงิน" icon="approve-check">
		ทำเครื่องหมายค่าใช้จ่ายหรือโครงการเป็นชำระแล้ว
	</Card>
</CardGrid>
