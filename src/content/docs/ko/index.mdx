## Evere 소개

Evere는 커플과 그룹의 더치페이와 정산을 손쉽게 관리하는 모바일 애플리케이션입니다. 다중 통화를 지원합니다.

<CardGrid stagger>
	<Card title="다중 통화 지원" icon="translate">
		여러 통화로 지출을 기록하고 자동 환산
	</Card>
	<Card title="유연한 분할" icon="puzzle">
		균등 분할, 비율 기반, 또는 수동 입력
	</Card>
	<Card title="프로젝트 관리" icon="suitcase">
		프로젝트별로 지출 정리
	</Card>
	<Card title="정산 추적" icon="approve-check">
		개별 지출 또는 전체 프로젝트를 정산 완료로 표시
	</Card>
</CardGrid>

## 지원 플랫폼

- iOS 15.1 이상
- Android 7.0 이상
