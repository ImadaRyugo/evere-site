**최종 업데이트**: 2025년 2월 3일

## 1. 소개

Evere("서비스")는 커플과 그룹의 더치페이와 정산을 손쉽게 관리하는 모바일 애플리케이션입니다. 본 개인정보 처리방침은 수집하는 정보, 사용 방법, 보호 방법을 설명합니다.

서비스를 이용함으로써 본 개인정보 처리방침에 동의하는 것으로 간주됩니다.

## 2. 수집하는 정보

### 2.1 계정 정보

다음 인증 방법을 제공합니다:

- **익명 인증(게스트 모드)**: 등록 없이 서비스 이용
- **Apple Sign In**: Apple ID로 로그인
- **Google Sign In**: Google 계정으로 로그인
- **이메일/비밀번호**: 이메일과 비밀번호로 인증
- **계정 연결**: 익명 계정에서 이메일 기반 계정으로 업그레이드 (전환)

이러한 방법을 통해 저장되는 정보:

- 사용자 ID (UUID)
- 표시 이름
- 강조 색상 (UI 설정)
- Pro 상태 (유료 기능 사용)
- 온보딩 완료 플래그

### 2.2 프로젝트 및 지불 데이터

핵심 기능으로 다음 데이터를 저장합니다:

- **프로젝트 정보**: 프로젝트 이름, 기준 통화, 참가자 이름, 정산 상태
- **지불 정보**: 제목, 금액, 통화, 카테고리, 지불 날짜, 메모, 분할 모드, 분할 금액/비율, 정산 상태
- **영수증 이미지** (선택사항): 지불 증명으로 이미지 첨부 가능
  - 이미지는 저장 효율성을 위해 최적화될 수 있습니다
  - 메타데이터(위치 정보 등)는 처리 중에 제거될 수 있습니다

### 2.3 기기 정보

푸시 알림 사용 시 수집:

- Expo Push Token (알림 전달용)
- 플랫폼 (iOS/Android/Web)
- 알림 활성화 플래그

### 2.4 로컬 캐시

오프라인 기능을 위해 기기에 로컬로 저장되는 정보:

- 환율 캐시 (TTL: 70분)
- 지불 데이터 (오프라인 동기화용)
- 인증 세션

## 3. 정보 사용 방법

수집된 정보를 다음 목적으로 사용합니다:

### 3.1 서비스 제공

- 더치페이 및 정산 기능
- 다중 통화 환산 (환율 조회)
- 프로젝트 공유 및 초대
- 오프라인 지원 (로컬 캐싱)

### 3.2 인증 및 접근 제어

- 사용자 인증
- 프로젝트 접근 관리

### 3.3 알림

- 프로젝트 초대 알림
- 지불 추가 알림

### 3.4 서비스 개선

- 오류 모니터링 및 충돌 보고서 (Sentry 사용)
- 성능 모니터링
- 세션 재생 (수집 범위 및 샘플링 비율은 설정에 따라 달라질 수 있습니다)

### 3.5 결제 관리

- 인앱 구매 (Pro 기능)
- 구독 상태 동기화

### 3.6 광고

- Google AdMob을 통한 광고 전달 (무료 버전만)
- 콘텐츠는 모든 연령에 적합합니다

## 4. 정보 공유 및 제3자 서비스

다음 제3자 서비스를 사용합니다:

### 4.1 Supabase (미국)

**역할**: 백엔드 인프라
**공유 데이터**: 인증 정보, 프로젝트/지불 데이터, 영수증 이미지, 기기 토큰
**목적**: 핵심 서비스 기능

### 4.2 Sentry (미국)

**역할**: 오류 모니터링 및 분석
**공유 데이터**: 오류 정보, 충돌 보고서, 성능 데이터, 세션 재생
**목적**: 앱 품질 개선

### 4.3 Expo Push Notification Service (미국)

**역할**: 푸시 알림 전달
**공유 데이터**: Expo Push Token, 알림 콘텐츠
**목적**: 프로젝트 초대 및 업데이트 알림

### 4.4 RevenueCat (미국)

**역할**: 인앱 구매 관리
**공유 데이터**: 사용자 ID, 구매 정보, 구독 상태
**목적**: Pro 기능 제공

### 4.5 Google AdMob (미국)

**역할**: 광고 전달
**공유 데이터**: 광고 식별자, 광고 표시/클릭 정보
**목적**: 무료 버전 수익화

### 4.6 ExchangeRate-API

**역할**: 환율 조회
**공유 데이터**: 통화 쌍 정보
**목적**: 다중 통화 환산
**접근 방법**: Supabase Edge Function을 통해 (시간당 1회)

### 4.7 OAuth 인증 제공업체

- **Apple**: Apple Sign In
- **Google**: Google Sign-In

## 5. 데이터 저장 및 삭제

### 5.1 데이터 저장 위치

- **Supabase Postgres**: 프로필, 프로젝트, 지불 데이터
- **Supabase Storage**: 영수증 이미지 (접근 제어된 저장소, 필요 시 시간 제한 URL을 통해 접근 가능)
- **로컬 기기**: 오프라인 캐시 (동기화 후 삭제)

### 5.2 데이터 보관 기간

- 활성 사용자: 서비스 사용 중 무기한
- 오프라인 캐시: 환율 70분, 지불 데이터는 동기화 후 삭제

### 5.3 데이터 삭제

- 영수증 이미지 삭제: 지불 편집 화면에서 가능
- 프로젝트 삭제: 프로젝트 설정에서 가능
- 계정 삭제: [support_email]로 문의

## 6. 보안

귀하의 정보를 보호하기 위해 다음 보안 조치를 구현할 수 있습니다:

- **HTTPS 통신**: 모든 통신이 암호화될 수 있습니다
- **Row Level Security (RLS)**: 데이터베이스 수준 접근 제어가 적용될 수 있습니다
- **Private Storage**: 영수증 이미지는 제한된 접근 방식을 통해서만 접근 가능할 수 있습니다
- **메타데이터 제거**: 이미지에서 위치 데이터가 제거될 수 있습니다
- **이미지 최적화**: 이미지가 효율적인 저장을 위해 최적화될 수 있습니다

## 8. 어린이 개인정보 보호

서비스가 모든 연령에 적합하도록 평가되었지만, 13세 미만 어린이로부터 의도적으로 개인정보를 수집하지 않습니다. 자녀가 개인정보를 제공했다고 생각되면 [support_email]로 문의하세요.

## 9. 쿠키 및 추적

### 9.1 웹사이트

공식 웹사이트는 현재 쿠키나 분석 도구를 사용하지 않습니다.

### 9.2 모바일 앱

Google AdMob은 광고 전달을 위해 광고 식별자를 사용할 수 있습니다. iOS에서는 광고 식별자의 사용 가능 여부가 사용자의 '앱이 추적을 요청하도록 허용' 권한 설정에 따라 달라집니다.

## 10. 개인정보 처리방침 변경

본 개인정보 처리방침은 법적 또는 서비스 변경으로 인해 예고 없이 업데이트될 수 있습니다. 중요한 변경사항은 인앱 알림 또는 이메일을 통해 알려드립니다.
