Evere는 커플이나 그룹의 더치페이·정산을 간편하게 관리할 수 있는 iOS 앱입니다. 여러 통화를 지원해, 여행부터 일상 생활비까지 앱 하나로 끝납니다.

이 가이드를 따라가면 Evere의 모든 기능을 자유롭게 쓸 수 있습니다. 먼저 기본 개념과 화면 구성부터 살펴볼게요.

## Evere의 4가지 기본 개념

이 네 가지만 알아 두면 나머지는 자연스럽게 이해됩니다.

- **프로젝트** — 정산의 '단위'입니다. 여행·데이트·동거처럼 상황마다 하나씩 만듭니다. 모든 지출은 프로젝트 안에 기록됩니다.
- **기준 통화** — 그 프로젝트의 정산을 어떤 통화로 계산할지 정하는 통화입니다. 프로젝트를 만들 때 선택합니다.
- **참가자** — 프로젝트를 함께 쓰는 멤버입니다. 앱을 쓰는 사람은 초대 링크로 참여할 수 있고, 앱이 없는 사람은 이름만으로 추가할 수 있습니다.
- **정산** — '누가 누구에게 얼마를 내면 되는지'를 Evere가 자동으로 계산합니다. 지출을 기록하기만 하면 정산 상태를 언제든 확인할 수 있어요.

## 화면 구성 (5개 탭)

Evere 하단에는 5개의 탭이 있습니다.

| 탭 | 역할 |
| --- | --- |
| **프로젝트** | 프로젝트 목록·생성·전환·설정 |
| **지출** | 선택한 프로젝트의 지출 목록 |
| **추가** | 지출을 기록 |
| **정산** | '누가 누구에게 얼마' 계산 결과·정산 상태 |
| **설정** | 계정·플랜·통화·알림 등의 설정 |

지출·정산 탭은 **현재 활성화된 프로젝트**의 내용을 보여 줍니다. 프로젝트 탭에서 다른 프로젝트를 선택하면 표시가 바뀝니다.

## 지원 환경

- **iOS 15.1 이상**의 iPhone에서 사용할 수 있습니다.
- Android 버전은 현재 제공하지 않습니다.
