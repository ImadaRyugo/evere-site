Evere는 커플과 그룹의 더치페이와 정산을 손쉽게 관리하는 모바일 애플리케이션입니다. 다중 통화를 지원합니다.

## 주요 기능

- **다중 통화 지원**: 여러 통화로 지출을 기록하고 자동 환산
- **유연한 분할**: 균등 분할, 비율 기반, 또는 수동 입력
- **프로젝트 관리**: 프로젝트별로 지출 정리
- **영수증 첨부**: 참고용 영수증 이미지 추가
- **정산 추적**: 개별 지출 또는 전체 프로젝트를 정산 완료로 표시
- **오프라인 지원**: 오프라인으로 작업하고 연결 시 동기화 (Pro)

## 시작하기

1. App Store에서 Evere 다운로드 (iOS)
2. 계정 생성 또는 게스트 모드 사용
3. 프로젝트 생성 및 참가자 초대
4. 지출 추가 및 분할
5. 정산 추적
