## Free 플랜과 Pro 플랜

Evere는 무료이며, 더치페이에 필요한 기능은 모두 쓸 수 있습니다. 제한을 없애거나 더 편하게 쓰고 싶다면 Pro를 이용하세요.

| 기능 | Free | Pro |
| --- | --- | --- |
| 프로젝트 수 | 3개까지 | 무제한 |
| 영수증 이미지 | 프로젝트당 5장 | 무제한 |
| 여러 통화·자동 환산 | ✓ | ✓ |
| 멤버 초대·실시간 동기화 | ✓ | ✓ |
| 수동 환율 | — | ✓ |
| 예산 설정·도달 알림 | — | ✓ |
| 오프라인 | — | ✓ |
| 광고 | 표시 | 숨김 |

:::note
가격과 구매 플랜(월간·연간·평생)은 앱에서 확인하세요. Pro에는 무료 체험이 제공되는 경우가 있습니다.
:::

### 업그레이드·구매 복원

- 업그레이드는 설정 탭의 '계정 관리' 또는 '플랜 상세'에서 할 수 있습니다.
- 기기 변경 등으로 구매를 이어받을 때는 '**구매 복원**'을 씁니다.
- 구독은 Apple ID에 청구되며, Apple 구독 설정에서 언제든 관리·해지할 수 있습니다.

## 문제 해결

### 정산 금액이 맞지 않거나 상대가 표시되지 않아요

**미확인 환율**이 있는 지출은 정산 계산에서 제외됩니다. 정산 화면에 경고가 있는지 확인하고, 해당 지출의 환율을 확정하세요([여러 통화와 환율](/ko/docs/currencies/#미확인-환율에-주의하세요)).

### 지출을 추가할 수 없어요

그 프로젝트가 **정산 완료** 상태일 수 있습니다. 프로젝트 목록에서 정산을 해제하면 다시 추가할 수 있습니다.

### 기준 통화를 바꿀 수 없어요

첫 지출을 기록하면 기준 통화는 변경할 수 없습니다. 다른 기준 통화를 쓰려면 새 프로젝트를 만드세요([프로젝트](/ko/docs/projects/#통화-설정)).

### 초대나 정산을 할 수 없어요

이 작업들은 계정 등록이 필요합니다. 게스트 모드에서는 할 수 없습니다. 설정 탭에서 계정을 등록하세요([계정과 데이터](/ko/docs/account/)).

### 기기 변경으로 데이터가 사라졌어요 / 사라질 것 같아요

게스트 모드의 데이터는 기기에만 저장됩니다. 기기를 바꾸기 전에 반드시 계정을 등록해 데이터를 클라우드에 저장하세요.

### 프로젝트를 4개 이상 만들고 싶어요 / 광고를 없애고 싶어요

이런 점은 Pro 플랜으로 해결할 수 있습니다(위 비교표 참고).

### 그래도 해결되지 않을 때

[문의 폼](/ko/support/)으로 연락 주세요. 앱의 설정 탭 '지원'에서도 문의할 수 있습니다.
